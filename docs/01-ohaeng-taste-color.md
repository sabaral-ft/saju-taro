name: 오행 맛·색깔 보충법
description: 부족한 오행은 해당 맛과 색으로 보충, 과다한 오행은 해당 맛 절제 — 구체적 매핑표
type: project
---

부족한 오행 → 해당 맛·색을 의식적으로 채움
과다한 오행 → 본능적으로 해당 맛을 과하게 찾으니 절제

| 오행 | 맛 | 색 | 부족 시 | 과다 시 |
|------|-----|-----|---------|---------|
| 목(인,묘) | 신맛 | 청색(푸른색) | 신맛 음식(식초,레몬,매실) 챙김, 청색 옷·소품, 산·바다 | 신맛 줄이기, 음주·야식 절제 |
| 화(사,오) | 쓴맛 | 적색(붉은색) | 쓴맛 음식(커피,녹차,씀바귀) 챙김, 붉은 밝은 옷 | 쓴맛(커피·차) 줄이기 |
| 토(진,술,축,미) | 단맛 | 황색(노란색) | 단맛 음식(꿀,고구마,대추) 챙김 | 단맛 지나치게 선호→줄여야 |
| 금(신,유) | 매운맛 | 백색(흰색) | 매운맛 조금씩 연습(매운맛에 약한 경향), 흰색 옷 | 매운맛 줄이기 |
| 수(해,자) | 짠맛 | 흑색(검은색) | 짠맛 음식(해조류,된장) 보강, 검은색 옷·소품 | 짠맛 줄이기 |

## 화(火) 보충 구체적 식재료 (한습 체질용)
- 쓴맛: 쑥, 더덕, 도라지, 씀바귀, 고들빼기, 여주, 민들레, 칡, 머위, 우엉, 쑥갓, 수수, 은행, 다크초콜릿(70%+), 커피, 녹차
- 차류: 영지버섯차, 홍차, 칡차, 자몽차 (따뜻하게 마실 것)
- 붉은색: 토마토, 팥, 홍고추, 대추, 구기자, 석류, 오미자, 당근, 비트, 소고기, 사과, 딸기, 체리, 홍파프리카, 크랜베리, 붉은강낭콩
- 조리법: 날것(샐러드)보다 따뜻하게 열을 가해 조리 권장

**Why:** 사주의 부족/과다 오행을 알면 일상 식단·패션에서 건강을 보완할 수 있음
**How to apply:** 대운/세운 건강 조언에서 부족 오행은 맛·색 보충, 과다 오행은 맛 절제 조언 포함
