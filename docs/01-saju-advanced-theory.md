## 0. 음양오행(陰陽五行) — 사주명리학의 근본 원리

### 음양(陰陽)
- **양(陽)**: 위로 올라가는 가볍고 맑은 기운 — 태양, 밝음, 따뜻함, 앞쪽, 적극적 움직임(動)
- **음(陰)**: 아래로 가라앉는 무겁고 탁한 기운 — 달, 어둠, 차가움, 뒤쪽, 고요함(靜), 수용적 성질
- 절대 분리 아님, 항상 함께 존재하며 대대(待對) 관계

### 오행(五行) — 5원소
목(木, 나무), 화(火, 불), 토(土, 흙), 금(金, 쇠), 수(水, 물)
고정이 아닌 끊임없는 순환 (상생·상극)

### 상생(相生) — 서로 낳고 도와주는 윈윈 관계
- **목생화**: 나무가 땔감이 되어 불을 만듦
- **화생토**: 불이 타고 재가 되어 흙이 됨, 태양 열이 흙을 단단히
- **토생금**: 흙/산 속에서 광물(쇠)이 잉태·탄생
- **금생수**: 바위 틈에서 지하수 솟음, 쇠가 녹아 액체(물)
- **수생목**: 물이 나무의 생명력·성장 도움

### 상극(相剋) — 서로 이기고 제압·통제하는 관계
- **목극토**: 나무가 흙에 뿌리 내려 수분·양분 빨아쓰고 흙을 갈라지게
- **토극수**: 흙이 제방/댐 형태로 물의 흐름을 막고 가둠
- **수극화**: 물이 불의 열기를 식히고 꺼버림
- **화극금**: 불의 열기로 단단한 쇠마저 녹여버림
- **금극목**: 쇠(도끼/톱/낫)로 나무를 베고 쪼갬

### 핵심 통찰
- 생(生)해준다고 무조건 좋은 것 아니고, 극(剋)한다고 무조건 나쁜 것 아님
- 영원한 강자/약자 없음
- 예: 물이 과다할 때 흙(토)이 극하여 댐 역할 → 유용 / 쇠(금)가 많아도 흙(토)이 뒤에서 생해주지 않으면 뒷심 상실
- 사주팔자 8글자 안에서 음양·오행의 상생상극 균형·조화(조후)를 분석 → 성격, 잠재력, 길흉화복 예측

## 0-0.5 만세력으로 일간 찾는 방법 (사용자 가이드)
1. 만세력 접속 (웹사이트 검색 또는 무료 앱)
2. 기본 정보 입력: 성별, 생년월일, 양력/음력 구분, 윤달 여부
3. 태어난 시간 입력 (사주 정밀도 핵심. 모르면 '모름' → 시주 제외 6글자로 한계)
4. 야자시(夜子時) 설정: **야자시 미적용(정자시법) 권장**
   - 정자시법: 밤 23:30 넘으면 다음 날 일주 적용
   - 야자시: 밤 23:30~0:30 사이 태어난 당일 일주 그대로 적용
   - 조자시(朝子時): 0:30~1:30 → 하루가 지난 것으로 계산
5. 결과에서 8글자(시간 모를 시 6글자) 사주 원국 → **일주(日柱) 윗글자 = 천간(天干) = 일간(日干)** = 나 자신

### 태어난 시간이 중요한 4가지 이유
1. **절기 교체 시점**: 연/월 기준은 양력/음력 1일이 아닌 절기의 정확한 시·분. 같은 날이라도 절기 전후 몇 분 차이로 연주/월주 전체 변경 → 전혀 다른 사주
   - **사례1 (입춘)**: 2020.2.4 입춘 18시 2분 → 18:02 이후 출생 = 경자년(庚子) 무인월(戊寅) / 18:00 출생 = 기해년(己亥) 정축월(丁丑). 단 2분 차이로 연주+월주 모두 변경
   - **사례2 (입동)**: 2020.11.7 입동 시각 기준 → 이전 = 병술월(丙戌) / 이후 = 정해월(丁亥)
   - **사례3 (청명)**: 청명 시각 기준 → 이전 = 신묘월(辛卯) / 이후 = 임진월(壬辰)
   - 연주는 입춘(立春), 월주는 각 월의 절기(입동·청명 등) 정확한 시·분 기준으로 교체
2. **자정 무렵 일주 결정**: 23:30 넘으면 정자시법으로 다음 날 → 분 단위 차이로 일주 자체가 바뀜
3. **사주팔자 완성 (시주)**: 시간 없으면 8글자 중 2글자 빠진 6글자 → 상생상극·합충의 전체 구조 파악 불가, 정밀 분석 한계
4. **말년운·자식운**: 시지(時支) = 인생의 열매(實), 61~80세 말년기 운세·심리 + 자식궁(子息宮) = 자식 덕·자손 관계 핵심 지표
5. **사주 원국 오류 위험**: 절기 교체일이나 자정 무렵 출생 시, 시간 없으면 6글자 자체가 본인 것인지 확신 불가 (연주/월주/일주가 다른 글자일 수 있음)

### 6글자 사주의 성격 파악 범위와 시주 부재 한계

**6글자만으로 충분한 부분** (핵심 성격·본성):
- 일간(日干): 나 자신, 외향적·주체적 성격
- 월지(月支): 타고난 본성, 무의식의 가치관 뿌리
- 일지(日支): 최종 판단·결정 시 작용하는 핵심 심리
- → 타고난 기질, 대인관계 방식, 감성·이성 발달 정도 등 핵심 심리 구조 파악 가능

**시주 없으면 놓치는 부분**:
- **체(體)·용(用) 관계**: 일주(日) = 본체[체], 시주(時) = 실제 쓰임[용]. 시주 없으면 일지의 결정 심리는 알아도 **실생활에서 어떻게 응용·표출하는지**(구체적 행동 양식) 파악 불가
- **보조 심리**: 시주 간지 = 일지가 최종 결정 내릴 때의 보조적 성격

**태어난 시간 모를 때 역추론 방법**:
1. **시주 성격 대조법**: 유력한 시간대별 시주를 세워보고, 각 십신·오행 성향이 명주의 실제 행동 양식·보조 심리와 일치하는지 대조 → 가장 맞는 시간 역추론
2. **표준시 보정**: 한국은 일본 동경시 표준시 사용 → 실제 자연 시간과 약 32분 차이. 출생 시간에 ~30분 더해야 정확한 시진(時辰) 도출 (예: 자시 = 23:30~01:30 적용). 출생 지역별로도 세밀 감안
3. **과거 사건 역추적법** (실전 상담에서 가장 널리 사용): 명주가 과거 겪은 굵직한 사건사고 발생 연도를 각 시간대별 대운/세운 흐름과 대조 → 가장 부합하는 시주 특정
4. **신체·환경적 보조 지표**: 잠자는 자세, 정수리 가마 위치·개수, 형제자매 수 등으로 보조 유추
- **말년기 심리(61~80세)**: 시지 = 꽃 진 후 '열매(實)'. 노년기 가치관 변화·지향 심리 예측 불가

**시주 없이 대운(大運) 예측 — 가능한 것과 한계**:
- **대운 도출 공식에 시주 불필요**: 순행/역행 = 년간 음양 + 성별로 결정, 대운 글자 = 월주 기준 순서 전개, 대운수(교운 나이) = 일간~절기 일수 계산. 연·월·일 정보만 사용 → 시주 몰라도 10년 주기 운세표 완벽히 산출 가능
- **대운 방(方)으로 굵직한 길흉 예측 가능**: 동방(木)·남방(火)·서방(金)·북방(水) 흐름을 6글자에 대입 → 유리/불리한 환경 시기, 인생 성패 큰 틀 파악 가능
- **한계 — 말년운·자녀 사건 정밀 예측 불가**: 대운은 사주 원국과 합충 상호작용으로 사건 생성 → 시주 2글자 없으면 대운이 말년(61~80세)·자식궁에 미치는 영향 파악 불가. 6글자로는 무난해 보여도 숨겨진 시지와 대운이 충(沖)하면 자녀 문제·말년 돌발 사건 발생 가능

## 0-0.6 천간(天干) vs 지지(地支) — 5가지 차이점

| 구분 | 천간(天干) | 지지(地支) |
|------|----------|----------|
| 위치·상징 | 하늘의 맑고 가벼운 기운, 윗글자 (10개: 갑을병정무기경신임계) | 땅의 무겁고 가라앉는 기운, 아랫글자 (12개: 자축인묘진사오미신유술해) |
| 이상 vs 현실 | 정신적 영역, 이상, 명분·목표 | 현실적 영역, 구체적 환경, 실제 사건 |
| 성격 | 동(動)적, 밖으로 드러나는 외향적 성격 | 정(靜)적, 겉으로 안 드러나는 내면 본성 |
| 일의 해석 | 일의 배경·상황·환경 조건 (겉모습) | 실질적 결과·결론 (실제 득실) |
| 시간·방위 | 구체적 시간/방향성 없음 | 시간(자정/정오), 방위(동서남북), 사계절 + 지장간(천간 2~3개 내장) |

**요약**: 천간 = 생각·의지·이상의 정신적 설계도 / 지지 = 그 생각이 구현되는 구체적 현실과 결과물

## 0-0.7 월지(月支) — 타고난 본성과 내면 심리의 뿌리

- 월지 = 부모님으로부터 물려받은 타고난 본성, 성격 형성에 가장 근본적·깊은 영향
- 천간(天干) = 밖으로 드러나는 외향적 성격 / 지지(地支, 특히 월지) = 무의식에 깔린 숨겨진 성격·심리의 뼈대
- 일간이 '나 자신의 주체적 기운'이라면, 월지는 '가치관과 내면 심리를 결정짓는 환경적 뿌리'

### 월지 십신별 타고난 본성 (일간↔월지 생극제화 기준 5그룹)
- **비겁(비견·겁재)** — 월지=일간과 같은 오행: 주체성·독립심 강함, 평등 의식, 강한 고집(아집)
- **식상(식신·상관)** — 일간이 월지를 생(生): 감성·융통성 발달, 낙천적·창의적, 언변·표현력 뛰어남
- **재성(편재·정재)** — 일간이 월지를 극(剋): 현실 감각 강함, 치밀·계산적, 성과·효율 중시
- **관성(편관·정관)** — 월지가 일간을 극(剋): 원리원칙·책임감·명예욕, 조직 적응력, 바른 본성
- **인성(편인·정인)** — 월지가 일간을 생(生): 수용성 강함, 사려 깊고 학문적, 직관·이성적 사고

### 월지 지장간과 사령(司令)
- 월지 안에 2~3가지 천간 기운(지장간: 여기, 중기, 정기)이 숨겨져 있음
- **사령** = '맡을 사(司) + 하여금 령(令)' = 지장간 중 실질적으로 당령하여 일을 주관하는 가장 왕(旺)한 기운
- **핵심 중의 핵심**: 월령이 사주 전체를 관장하는 핵심이라면, 사령은 그 핵심의 중추. 사령 간과 = 간명(看命)의 핵심을 빼놓는 것
- **길흉화복 결정**: 길신으로 쓰이면 크게 발복(發福), 흉신으로 쓰이면 대패(大敗)의 주요 원인
- **내면 심리**: 월지=타고난 본성, 천간 투출 용신=겉으로 드러나는 사회성이라면, 사령=본성 속 깊이 자리잡은 은밀한 내면 심리. 이 기운에 가장 의지하고 일을 맡기려는 심리 강하게 작용

## 0-0.8 일주(日柱) — 사주의 가장 핵심적인 기둥

일주 = 태어난 날의 간지 = 일간(日干, 위) + 일지(日支, 아래)

### 1. 일간(日干) — 나 자신의 주체적 기운
- 사주의 주인공 = '나 자신(명주)'
- 현대 신법 명리학(자평학): 일간을 절대적 기준으로 나머지 글자와의 상호작용(길흉화복, 격국) 분석
- (과거 고법 명리학은 연주=조상궁 위주)

### 2. 일지(日支) — 배우자궁 + 최종 결정 심리
- **배우자궁**: 아내/남편 상징. 길신이 자리잡고 일간의 뿌리가 되면 배우자덕·부부덕 있음
- **최종 결정 심리**: 일간과 가장 가까이 늘 함께하는 자리 → 명주가 어떤 일을 최종 판단·결정할 때 작용하는 핵심 심리

### 3. 일지 — 장년기(41~60세)와 인생의 꽃(花)
- 사주 네 기둥의 인생 흐름 중 장년기 운세·심리 기준
- 식물 비유: 꽃이 만개하는 시기. 일지에 길신 → 평탄하고 좋은 장년기

## 0-1. 일간(日干) 10천간 — 타고난 본성과 성향

일주(日柱) 천간 = 나 자신을 상징하는 핵심 글자

### 목(木) — 인(仁), 어질고 위로 솟구치며 시작하는 기상
- **갑(甲) 양목**: 큰 나무. 장엄하고 위로 상승하려는 기상. 학자/교육자처럼 양반스럽고 꼿꼿한 성향
- **을(乙) 음목**: 봄 새싹, 덩굴, 꽃. 겉은 부드러우나 억세고 끈질긴 생명력. 내적 성숙 추구, 유연한 성향

### 화(火) — 예(禮), 밝게 확산하는 기운
- **병(丙) 양화**: 태양. 만물을 기르는 빛, 따뜻한 배려심. 밝고 강렬한 열정
- **정(丁) 음화**: 화로불, 촛불, 용접 불꽃. 고요하지만 강한 집중력, 파고드는 연구심리, 이성적·현명

### 토(土) — 신(信), 신용과 믿음, 포용력과 중재
- **무(戊) 양토**: 큰 산, 거대한 댐, 넓은 대지. 변함없고 푸근하게 만물을 품어줌. 고정된 진리 추구, 듬직한 관리자
- **기(己) 음토**: 정원, 논, 밭, 옥토. 안정적이고 지키며 기르는 중간 관리자, 수용적

### 금(金) — 의(義), 옳고 그름 분별, 결단하는 의리
- **경(庚) 양금**: 큰 바위, 쇳덩어리. 웅장하고 굳센 강건함. 단숨에 결단, 개혁적·장렬한 기상
- **신(辛) 음금**: 보석, 반지, 주사기, 칼. 치밀하고 예리. 냉철한 판단력, 이성적 사고

### 수(水) — 지(智), 지식과 지혜, 저장·응축
- **임(壬) 양수**: 큰 바다, 호수. 기운 수렴, 은은히 퍼짐. 잔잔하고 평온하나 속을 모르는 깊은 성향
- **계(癸) 음수**: 시냇물, 샘물, 안개, 이슬. 호기심·탐구심 강함, 지략과 지모, 영민하고 애교 있는 성향

### 4. 십신(十神) 상세 심리 — 5그룹 10신 장단점

**① 비겁 (주관과 의지력)**
- **비견**: 강한 주체의식·자존심, 평등·인화·협동 중시. 단점: 자기주장·아집, 독불장군
- **겁재**: 불굴의 기상, 강한 의지력·도전정신, 야심가. 단점: 지기 싫은 고집, 때·장소 안 가리는 과욕

**② 식상 (감성과 표현력)**
- **식신**: 낙천적, 정 많고 배려, 시비 가려내는 능력, 예술성. 단점: 식탐, 먹고 즐기기 좋아함
- **상관**: 두뇌 회전 빠름, 임기응변·창작·표현력 최상, 패기·자유분방. 단점: 급한 성격, 변덕, 오만, 규범 파괴

**③ 재성 (현실 감각과 결과 지향)**
- **편재**: 시원시원, 화통, 손 큰 사업가 기질, 눈치·판단력·결단력. 단점: 기분에 좌우, 헤픈 씀씀이, 인간미 상실
- **정재**: 치밀·꼼꼼·계산 빠름, 알뜰·성실·안정감. 단점: 소심, 따지기, 결벽증, 안주

**④ 관성 (원칙과 명예)**
- **편관(칠살)**: 청렴결백, 원칙 중시, 직관력·예리함, 인내력. 단점: 냉혹·단호, 한순간 단절
- **정관**: 공명정대, 원리원칙, 따뜻한 관리자, 믿음직. 단점: 체면 의식, 복지부동, 융통성 결여

**⑤ 인성 (사고력과 수용성)**
- **편인**: 철학적·신비적 관심, 상상력·창의력, 두뇌형. 단점: 겉으로 듣지만 속으로 의심, 생각만 앞서고 행동 귀찮아하는 게으름
- **정인(인수)**: 엄마처럼 푸근, 사려 깊음, 도덕·도리 중시, 수용적·어진 성품. 단점: 보수적, 안정만 추구, 고리타분, 무턱대고 가르치려 함

**과유불급**: 특정 십신이 없거나 너무 많으면 해당 십신의 단점이 극단적으로 발현 (융통성 부족, 이기적 성향)

### 4-1. 십신(十神) vs 육친(六親) 차이

**십신** = 일간 기준으로 다른 글자와의 관계를 **추상적 기운(에너지)**으로 분류한 것. 내면 심리·성향·적성 파악 기준 (10가지)
**육친** = 직계가족 6인(부·모·형제·자매·처·자) + 확대 대인관계에 십신을 **구체적 인물**로 대입한 것

| 십신 그룹 | 남성 기준 육친 | 여성 기준 육친 |
|-----------|---------------|---------------|
| 비겁(비견·겁재) | 형제·친구·동료·경쟁자 | 형제·친구·동료·경쟁자 |
| 식상(식신·상관) | 아랫사람·후배·부하 | 자식(아들·딸)·아랫사람 |
| 재성(편재·정재) | 부인·부친·재물 | 시어머니·재물 |
| 관성(편관·정관) | 자식(아들)·상사·직장 | 남편·직장·상사 |
| 인성(편인·정인) | 부모·조부모·스승 | 부모·조부모·스승 |

**핵심 차이**: 십신은 "어떤 기운인가"(심리·성향), 육친은 "누구인가"(인물·관계). 같은 정재라도 십신으로 보면 '안정적 재물 관리 심리', 육친으로 보면 남성에겐 '아내', 여성에겐 '시어머니'

### 5. 지장간 심리 3단계 구조
- 월지: 타고난 본성(本性) — 고정
- 투출 용신: 사회성/페르소나 — 겉
- 사령: 가장 깊은 내면 심리 — 속
- 투출 시: 생각=행동 일치
- 미투출 시: "가만히 쓸 때를 기다림" (자평진전)

### 6. 합충(合沖) 심리 분석 (궁위별)

**충(沖) — 충돌과 갑작스러운 변화**
- 기운이 7번째 만나 충돌 → 다툼·시시비비 발생 쉬움
- 기운을 역동적으로 움직이게(動) 하므로 부지런하고 활동적 성향
- 사건: 자신의 의지와 상관없이 **갑작스럽게 발발**하는 돌발 사고·예기치 않은 사건
- **궁위별 충의 구체적 변화**:
  - 년지(조상궁) 충: 외국/먼 타지행, 조부모·조상 묘지 이상
  - 월지(부모궁) 충: 직장 이동, 부모 곁 떠나 독립
  - 일지(배우자궁) 충: 이사, 배우자 불화·충돌
  - 시지(자식궁) 충: 자녀가 곁을 떠남, 자녀 문제 발생
- 천간충 = 심리적 배경·환경 변화 / 지지충 = 현실적 결론·결과

**합(合) — 결속과 계획된 상황**
- **현대 자평명리학 해석**: 본래 두 기운이 새 오행으로 변화(진오행)하는 것이나, 현대에는 기운이 묶여 **본래 기능이 일시 정지/묶임**으로 주로 해석
- **삼합(三合)** 심리: 생·왕·고 3기운 결합 → 광범위한 합의·타협 능력 탁월. 단, 합다자(合多者)는 대의/불의에도 쉽게 타협 → 소인배적 기질 우려
- **육합(六合)** 심리: 음양 1대1 결합 → 남녀관계·개인 간 긴밀한 합의, 대인관계 능력 발현
- 사건: 충과 달리 미리 **계획/준비한 일이 순조롭게 실현**
- 합 과다 시: 화합·타협 지나쳐 우유부단, 소인배적 심리
- **통변 예시**: 남자 사주 일지(배우자궁)에 재성(財星) 운이 합(合) → 이성 만남 성사, 여자와 즐거운 시간을 보내려는 시기로 해석

**통변 핵심 원리**: 합충이 사주의 **어느 기둥(궁위)**과 **어떤 십신**에 작용하는지를 종합해 그 해의 길흉화복을 구체적으로 예측

**결론**: 충 = 예기치 못한 환경 변화·역동적 이동·관계 충돌 / 합 = 관계 결속·타협·계획된 일의 실행·기운 묶임

**합충 동시 발생 시 상호작용**
- 합+충 공존 → 상반된 기운 견제, 매우 변화무쌍하고 복합적 상황
- **탐합망충(貪合忘沖)·선합후충(先合後沖)**: 합을 탐하느라 충의 작용을 잊음 → 충돌 상황임에도 타협·결속이 먼저 이루어져 충의 파괴력 약화/갈등 무마
- **충발(沖發) — 합을 깨는 충**: 원국에서 합으로 기능 정지/안정된 상태에 강력한 충 가해짐 → 묶인 기운 풀리며 급격한 이동, 관계 단절, 억눌린 잠재력 폭발
- **관계 통변 예시**: 합충 불안정 얽힘 → 연애는 많으나 결혼 결실 못 맺음 등 변동성 큰 요인
- **십신별 영향**: 식신(食神)이 합충 겪으면 체격 왜소해지거나 소견 편협해지는 등 성향·외형 변화
- **결론**: 갈등 속 타협 안정 or 계획된 일이 돌발 변수로 엎어지는 굴곡진 변화

**합충 동시 발생 실전 통변 사례 3가지**
1. **주거지+회사 동시 이전**: 술토(戌) 운 → 월지 진토(辰)와 진술충(辰戌沖) + 일지 묘목(卯)과 묘술합(卯戌合) → 충(기존 환경 타파·이사) + 합(새 계획 실행) 동시 작용 → 거주지 이사 + 회사 이전
2. **먼 곳 이주 결심**: 신금(申) 월운 → 년지 진토(辰)와 신진반합(申辰半合) + 일지 인목(寅)과 인신상충(寅申相沖) → 반합(새 계획 묶임) + 상충(갑작스러운 이동) → "먼 곳으로 이사 가서 먹고살 생각 모색"
3. **원거리 직업 이동+자식 다툼**: 자수(子) 월운(정관) → 년지 진토(辰)와 자진반합(子辰半合) + 시지(자식궁) 오화(午)와 자오충(子午沖) → "먼 곳 직업 이동 + 자식 관련 다툼 동시 발생"

**사례 공통 패턴**: 충 = 기존 환경 타파·이동 / 합 = 새 계획 실행·관계 형성 → 동시 맞물려 다이내믹하고 복합적 사건 발생

**합충 불안정 시기 대처법 5가지**
1. **신체 전조증상 파악**: 나쁜 운·큰 변동 다가올 때 소화불량·변비 등 몸이 먼저 신호 → 건강 상태 점검·사전 대비
2. **사전 예측·멘탈 관리**: 좋은 운은 활용 준비, 불리한 운은 보완책 미리 파악. 혼란스러울수록 롱런 위한 멘탈 관리 우선
3. **주체적 개입**: 현대 명리학 = "주어진 운명을 어떻게 활용할 것인가"의 실용 학문. 길흉에 일희일비 말고 변화를 스스로 통제·방향 모색
4. **긍정적 인연·공덕 쌓기**: 같은 사주라도 인연과 태도가 미래 결정. 인복은 후천적 성향 강함 → 먼저 베풀면 위기 시 귀인으로 돌아옴
5. **부족한 오행 기운 보충**: 오행 불균형 해소 위해 색상(청·적·백·흑)의 옷, 맛(신·단·쓴·매운)의 음식을 의도적으로 섭취·착용하여 밸런스 조절

### 7. 대운 방(方)과 일간 상생상극 로직
- 대운 방의 오행 → 일간 기준 십신 변환 → 용신/기신 여부 판단
- 일간을 돕는 방 = 길(吉), 흉신을 살리는 방 = 흉(凶)
- 월지(격국)와 일간 중심 2중 교차 분석

### 8. 투출/미투출 개운법
- 투출: 에너지를 긍정적으로 소화할 직업/환경 선택, 겸격은 중화(中和) 지향
- 미투출: "가만히 쓸 때를 기다림", 내면 정화, 운 도래 시 기회 포착 준비
- 공통 개운 3요소: 공부/전공 + 직업 환경 + 배우자 인연
- **미투출 심층 개운법 3단계**:
  - (1) 은둔자(The Hermit)의 지혜: 억압된 감정/욕구를 정서적 안전망 속에서 점진적 탐색·수용 → 내면 정화의 첫걸음
  - (2) 운명결정론 탈피: 사주를 '확정성'이 아닌 '가능성의 탐색'으로 인식, 운 도래 시를 위해 실력 연마·잠재 가능성 확대
  - (3) 후천적 3대 요인의 주체적 선택:
    - 공부/전공: 부족한 오행을 채워줄 학문 선택
    - 직업 환경: 잠재력을 안전하게 발산할 무대 조성
    - 배우자: 넘치거나 부족한 기운을 상호 보완할 인연
  - 결론: 타고난 복분(福分) 범위 안에서 최대한의 길운을 누리도록 삶을 개척

### 8-1. 사령 대운 투출 시 3대 변화 (핵심 전환점)
- (1) 억눌린 본심의 폭발적 표출: 오래 감춘 진짜 욕구를 과감히 분출 → 주변인 "사람이 완전히 달라졌다"
- (2) 삶의 방향·가치관 전면 전환: 기존 순응적 가면 벗고, 진짜 원했던 방향으로 삶의 궤도 전면 수정
- (3) 기존 천간과 생극제화에 따른 성패 결정:
  - 길신 작용 → 준비했던 잠재력 발현 → 발복(發福)·대성취
  - 흉신 작용 → 용신 파괴/합거 → 대패(大敗)·치명적 실패
- 상담 핵심: 투출 시기에 맹목적 분출 경계, 중화(中和)로 승화 조언

### 8-1a. 사령 길신 작용 시 십신별 발복 패턴
- **비겁 길신**: 주관·추진력 긍정 발휘 → 동업자/형제/지인의 결정적 도움 → 큰 투자 성공, 횡재
- **식상 길신**: 억눌린 재능·창의력 발현 → 기발한 변화로 큰 인기 → 의식주 안정, 부하/자식 덕
- **재성 길신**: 현실감각·사업 판단력 극대화 → 큰 부(富) 축적 → 부친 혜택, 현명한 배우자 덕
- **관살 길신**: 난제 완수 → 타인 존경 → 고속 진급, 최고 위치(권위) 영예
- **인성 길신**: 깊은 사고력·수용력 → 학문 성취, 시험 합격, 문서(부동산) 획득 → 부모/윗사람 조력
- 결론: 사령 길신 = 내면 갈망과 사회적 현실의 완벽 조화 → 타고난 최고 무기로 대성공

### 8-2. 사령 흉신 작용 시 십신별 파괴 패턴
- **비겁 흉신**: 맹목적 고집/과욕 → 투쟁·쟁탈 심리 → 파산, 지인 고통
- **식상 흉신**: 식신=싫증/무기력, 상관=법규 경시/파괴적 반항 → 먹고살 길 차단, 직장·명예 상실
- **재성 흉신**: 극단적 물질 추구 → 무리한 물욕 발동 → 부도/가산 탕진, 부친·배우자 갈등
- **관살 흉신**: 칠살 특유의 극단적 억압 → 극한 스트레스 → 관재수/해고/수술/질병
- **인성 흉신**: 그릇된 망상·맹목적 고집 → 현실 판단력 상실 → 시험 낙방/해고, 윗사람 번민
- 결론: 사령 흉신 = 가장 의지하던 내면 심리가 자신을 찌르는 무기로 전환 → 경거망동 경계 필요

### 8-3. 흉신 사령 제어 및 승화 (역용의 원리 + 융 그림자 이론)
- **역용(逆用)**: 길신은 순응·보호, 흉신은 역으로 제어해야 함
  - 충(沖)/극(剋)으로 제어, 합(合)으로 기능 정지(합거), 설기(洩氣)로 기운 빼내기
- **융(Jung) 그림자(Shadow) 통합**: 흉신 사령 심리 ≈ 무의식 그림자
  - 악으로 규정·제거가 아닌, 직면·수용·자아 통합이 핵심
  - "자신의 열등을 받아들이는 것 = 자신을 사랑하는 법을 배우는 위대한 철학"
- **승화 직업군 (후천적 개운)**:
  - 칠살(편관) 사령 → 의료/경찰/군인/검찰/종교 (활인 분야)
  - 상관 사령 → 음악/예술/방송/발명 (틀을 깨는 창의 분야)
  - 겁재/양인 사령 → 무역/개척/경호 (험난한 환경의 경쟁적 직업)
- **개운 3요소**: 공부(전공) + 직업 환경 + 배우자 → 선천적 약점 보완
- 흉신은 제대로 다듬으면 타의 추종 불허하는 강력한 무기로 전환 가능

### 9. 실전 통변 사례 (사용자 제공)
- 75년생 남: 상관견관을 인운이 억누름 → 직장 안정
- 72년생 여: 재물 합거 + 겁재 세운 → 파산
- 66년생 여: 인운에 억눌린 특기가 칠살·상관 발동으로 개화
- 75년생 여: 정관+비견 합 → 융통성 회복, 직장·가정 화합
- 76년생 남: 진술충+묘술합 → 이사+회사 이전
- 64년생 남: 정임합 → 용신 합거 → 퇴직
- **75년생 여 (길신 발복)**: 월지 정관=용신, 인운→식상운 순탄, 무술년 정관+세운 합 → 직장 화합·가정 평안 (국립대 교수)
- **66년생 여 (길신 발복)**: 월지 식신 본성, 젊은 시절 인운에 억눌림 → 병술대운 기해년에 칠살·상관 긍정 발동 → 어린이집 자영 성공

### 10. 12운성(十二運星) 상세 심리·생명력 해석
인간의 삶을 탄생~죽음~윤회의 12단계로 표현. 단순 운의 강약이 아닌 심리적 성숙과 생명력 변화 과정.

1. **양(養)**: 태아가 뱃속에서 보호받으며 자라는 시기. 내적 균형·성숙을 다지며 생을 준비하는 고요하고 수용적인 단계
2. **장생(長生)**: 세상 밖으로 첫 발을 내딛는 출생. 새로운 가능성과 성장의 출발점, 순수함, 자연스러운 인덕과 사랑
3. **목욕(沐浴)**: 출산 후 양수와 피를 씻어내는 정화. 외부 세계 첫 접촉, 타인 시선 의식, 감정 기복·자기표현 욕구 강함, 자유롭고 도전적
4. **관대(冠帶)**: 유년·청소년기. 자기주장 강하고 도전을 두려워하지 않음, 열정과 시행착오
5. **건록(建祿)**: 청년기, 신체·정신 가장 왕성. 자립·독립 에너지 최강, 책임감과 추진력
6. **제왕(帝旺)**: 12운성 정점, 장년기. 권위·영향력 최대, 역량 최대치 발현, 사회적 성공과 완숙
7. **쇠(衰)**: 노년의 시작. 제왕을 지나 기력 하강하나 삶에 대한 깊은 통찰력·정신적 여유 심화. 무리한 도전보다 역량 기반 현실적 안정 추구. 반안살과 연결되어 겸손·포용의 지혜로운 리더십 발휘, 안정적 성취. 에너지는 '약'이나 내면 지혜·덕망으로 무리수 없이 선한 영향력 행사
8. **병(病)**: 황혼기. 육체·정신 피로하나 아픔 통해 타인 공감·이해 심화, 내면 풍요 추구
9. **사(死)**: 삶 정리, 외적 활동 종결. 정신 각성의 내면적 전환점, 철학·종교·명상에 끌림, 삶과 죽음 경계 재정립
10. **묘(墓)**: 땅에 묻혀 정적 존재. 정리와 휴식, 재출발 위한 강한 내공 축적, 심신 회복기
11. **절(絶)**: 모든 인연·욕망 단절. 완전한 소멸, 집착·욕망 비움, 다시 시작 준비하는 근원적 정화

**⚠️ 사(死)·절(絶) 해석 주의 — 단순 '나쁜 운' 아님:**
- **사(死)**: 육체 정지 대신 정신 크게 각성. 철학·종교·명상·상담에 끌림, 삶의 본질 탐색하는 내면적 전환점
- **절(絶)**: 외적 극단적 변화 가능하나, 내적으로 모든 집착·욕망 비움 → 단절 속에서 진정한 자유 탄생. 완전한 비움을 통한 근본적 정화 에너지
- **올바른 접근**: 12운성은 고정 길흉이 아닌 생명력의 '변화 과정'. 사·절은 무리한 외부 활동을 멈추고 내면을 다지며 다음 전환점을 준비하는 시기로 해석
12. **태(胎)**: 전생 인연·경험 품은 영혼이 새 생명으로 잉태. 미완성이나 에너지 씨앗이 움트며 무한한 가능성, 새 출발 대기

**12운성 에너지(왕·중·약) 분류:**
- **왕(旺)** — 장생, 관대, 건록, 제왕: 신체·정신 에너지 가장 왕성, 자립·추진력·리더십 강하게 발현
- **중(中)** — 목욕, 묘, 태, 양: 내적 성숙·휴식·전환 준비의 중간 기운
- **약(弱)** — 쇠, 병, 사, 절: 육체·외적 에너지 크게 하강, 그러나 정신적 통찰·내면 각성 심화

**12운성 확인 방법:**
- 수동: 일간(태어난 날의 천간) 기준으로 지지를 12운성표에 대조. 예) 일간 갑(甲) + 지지 인(寅) = 건록, 갑(甲) + 묘(卯) = 제왕
- 각 기둥(연주·월주·일주·시주)별로 12운성 배치 확인 → 인생 단계별 흐름 입체적 파악

### 11. 십이신살(十二神殺) — 삼합 기반 지지 해석 체계
12운성(일간↔지지)과 달리 **지지↔지지** 관계에서만 발생, **삼합(三合)** 기반 해석.

**12가지 신살:** 겁살, 재살, 천살, 지살, 연살, 월살, 망신살, 장성살, 반안살, 역마살, 육해살, 화개살

**확인 방법:**
- 기준: 사주의 삼합 오행(목·화·금·수) 파악 후 십이신살 표와 지지 대조
- 예) 삼합 목(木): 자(子)=연살, 인(寅)=망신살, 묘(卯)=장성살
- 예) 삼합 수(水): 자(子)=장성살, 인(寅)=역마살 — 같은 글자도 삼합 오행에 따라 다른 신살

**일반 신살(양인살·괴강살 등)과 차이:** 글자 두 개의 직관적 관계가 아닌, 12개 기운 체계 속 종합적·구조적 해석 도구

**주요 신살 상세:**
- **역마살**: 활동적·이동 잦은 성향, 한곳 정착보다 돌아다니는 '떠돌이 인생' 기운
- **도화살(≒연살)**: 타인을 끌어당기는 매력·인기, 특히 이성적 매력 강하게 발산. 과거에는 부정적 요소로 여겨졌으나 현대에는 자기표현·인기가 중요해 사회적 성공을 이끄는 긍정적 요소로 재해석
- **화개살**: 화려함 + 뛰어난 집중력 + 깊은 내면 성찰. 역마살(외부 이동)·도화살(외부 매력 발산)과 대비되는 내면 집중·자기 성찰 기운. 적합 직업군: 종교인, 철학자, 심리상담가, 예술가, 학자, 문화예술 기획 — 혼자만의 시간 속 깊이 있는 연구·창작에 부합
- → 역마살(이동)·도화살(매력)·화개살(내면)은 십이신살의 대표적 3대 신살, 모두 삼합 기반 구조적 해석
- **반안살(攀鞍殺)**: '벼슬에 오름'의 긍정적 기운. 12운성 '쇠(衰)'와 맞물림. 특징: ①지혜로운 지도자적 면모(겸손·포용력·강한 내공) ②현실적 안정 추구(무리수 안 두는 지혜) ③명예와 편안함(말안장에 올라탄 모습처럼 안정적 성취). 치열한 경쟁 후 능력 인정받고 지혜·덕망 갖춰 안정적 성공과 여유를 누리는 기운

### 12. 대운(大運)과 세운(歲運) — 변화하는 운의 흐름
사주팔자(고정) = 배에 탄 사람의 체력/기술, 대운·세운(변화) = 바다의 파도와 바람

**대운(大運) — 인생의 큰 물결 (10년 주기):**
- 월주 기준 계산, 만 3~9세 첫 대운 시작, 10년마다 교체
- 대운 오행이 사주와 상호작용(합·충·보완)에 따라 10년간 전반적 삶의 기복 결정
- 10년 동안 놓이게 될 큰 환경과 무대

**세운(歲運) — 올해의 파도 (1년 주기):**
- 그 해의 간지(干支)에 의해 결정. 예) 2026년 = 병오년(丙午年) → 병(丙)+오(午) 에너지
- 대운 위에 겹쳐서 작용, 1년간 구체적 사건·변화

**대운×세운 결합 해석:**
- 대운 좋음 + 세운 좋음 = 최고의 해
- 대운 어려움 + 세운 나쁨 = 유독 힘든 해
- 대운 나쁨 + 세운 좋음 = 당해만큼은 돌파구 가능
- 사건 해석 시 반드시 대운+세운 결합이 필수

### 13. 12운성 vs 12신살 — 비교·상호보완 관계

**기준 차이:**
- 12운성: 천간(일간) ↔ 지지 비교 → 생로병사 윤회 단계
- 12신살: 지지 ↔ 지지(삼합 기반) → 12가지 외부적 기운

**해석 관점 차이:**
- 12운성 = 내면적 에너지 (생명력 전개 과정, 심리 성숙도, 에너지 강약의 '변화 과정' 자체에 초점)
- 12신살 = 외부적 현상 (구체적 사건·행동 특성·현상을 체계적으로 추론)
- 핵심 비유: 12운성이 "어떤 내면 상태와 체력(배경)"을 보여준다면 → 12신살이 "그 상태에서 어떤 외부 사건·특성(결과)"이 나타나는지 묘사 → 사주를 입체적으로 풀이

**의미상 교차점 (실전 해석):**
- **건록 ↔ 망신살**: 에너지 최왕성 시기의 자만심 → 망신 우려
- **쇠 ↔ 반안살**: 육체 하강 + 정신 통찰 심화 → 무리한 도전보다 현실 안정, 지혜로운 지도자 면모

**Why:** 사용자가 명리학 전문 지식을 기반으로 앱의 정확도를 높이고자 함
**How to apply:** 향후 합충 분석, 내방사주, 격국 전환, 12운성×12신살 교차 해석, 대운·세운 결합 로직 구현 시 이 데이터 참조
