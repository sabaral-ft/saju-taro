# 교차분석 규칙 — 모든 영역은 상호 연결

## 핵심 원칙
> "사주는 독립된 교과서가 아니라 거미줄처럼 엮인 웹이다"

## 1. 건강 → 모든 영역에 영향

### 건강 → 직업
- 화 부족(공황/불안) → "적극적으로 도전하세요" 금지 → "비대면/재택 권장"
- 수 부족(허리/신장) → "장시간 앉는 업무" 금지
- 금 부족(폐/호흡기) → "먼지 많은 환경" 금지
- **모순 금지**: "야간근무 피하라" + "유흥업종 추천" 불가

### 건강 → 재물
- 공황/불안 → "온라인 수익, 재택 부업" 추천
- 만성피로 → "파트타임, 체력 소모 적은 수익" 추천
- 신약 사주 → "무리한 사업 확장 자제"

### 건강 → 연애
- 공황/불안 → "비대면 만남부터, 소수 깊은 관계" 추천
- 화 부족 → "사람 많은 곳 피하라" → "소규모 만남"

### 건강 → 대운/세운
- 대운이 아무리 좋아도 건강이 안 좋으면 "에너지가 높아 도전!" 금지
- 대운×세운 이중억제 = 위험신호, 이중보충 = 황금기

## 2. 나이별 보정

### 0~12세
- 허리 아픔, 직장/재물 조언 불가
- 성장, 학습, 부모 관계 중심

### 13~19세
- 학업, 진로 탐색 중심
- 연애 조언은 가벼운 톤

### 20~59세
- 풀스펙트럼 (직업, 재물, 연애, 건강)

### 60세 이상
- 건강 비중 크게 높임
- "가능성이 싹트는 시기" → "건강 관리에 집중하며 지혜를 나누는 시기"
- "준비와 성장" → "몸과 마음의 안정이 가장 중요한 시기"
- "대폭 성장" → "건강이 뒷받침되면 작은 기쁨을 찾는 시기"

## 3. 오행 양면성 (부족해도 병, 지나쳐도 병)

### 부족한 오행
- weakestOhaeng → 해당 장기 취약 + 맛·색 보충 조언

### 과다한 오행
- dominantOhaeng → 해당 장기 과열/과부하 + 맛 절제 조언

### 상생/상극 양면성
- 상생이 무조건 좋은 게 아님 (과한 상생 = 에너지 유출)
- 상극이 무조건 나쁜 게 아님 (적절한 상극 = 규율·성장)

## 4. 신강/신약 교차

### 신강 사주
- 운동: 격렬한 운동 (축구, 격투기, 헬스)
- 음식: 과다 오행 맛 줄이기 + 식상 오행 맛 섭취
- 미발산 시: 오만, 고집, 급발진, 멘탈 붕괴
- 실내 발산: 수다, 글쓰기, 창작, 홈트

### 신약 사주
- 운동: 명상, 요가, 필라테스 (에너지 보존)
- 격투기/헬스 → 기가 눌려 건강 해침 (금지)
- 인성(배움, 독서) 채우기 필수

## 5. 재물 교차분석

### 재성 과다 ≠ 부자
- 일간이 감당할 힘(비겁) 있어야 진짜 부

### 수(水) 기운 = 저축 능력
- 수 부족 → 벌어도 새나감 경고

### 신왕재왕 vs 재다신약
- 재다신약: 인성(학문/멘토) 보강 필수, 재성운 때 건강 악화 경고

### 무재 사주
- 관성/인성으로 우회적 부 축적 추천
- 재성 대운 때 과감한 도전 + 감각 유지

## 6. 합충 → 건강/변동

### 지지충 → 장기 건강
- 자오충 → 심장(화)/신장(수) 충돌
- 인신충 → 간(목)/폐(금) 충돌
- 사해충 → 심장(화)/신장(수) 충돌

### 대운/세운 충 → 변동
- 일지 충 = 이사/배우자 변동
- 월지 충 = 직업/부모 변동

## 7. 조후(한난조습) 교차

### 한습 체질
- 우울증, 냉증, 무기력 취약
- 쓴맛(화)+매운맛(금) 보충, 따뜻한 환경
- 가벼운 유산소 운동

### 난조 체질
- 급한 성격, 심장과열, 건조 질환 취약
- 짠맛(수)+신맛(목) 보충, 시원한 환경
- 수영, 스트레칭

## 8. 타로 연계 교차

### 오행 → 수트 가중치
- 부족한 오행의 수트 정방향 = 필요한 에너지 (긍정)
- 과다한 오행의 수트 = 경고 증폭

### 건강 → 타로 톤
- 건강 약한 사람에게 "공격적으로 행동하라" 타로 해석 금지
- R.I.T.E. 원칙: 힘을 실어주되 현실적으로
