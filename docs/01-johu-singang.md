## 조후(調候) — 한난조습
- 한습(寒濕): 수+목 과다 → 우울증, 냉증, 무기력, 감정 동요
- 난조(暖燥): 화+금 과다 → 급한 성격, 폭발, 심장과열, 건조 질환
- 조후 균형이 잡힌 사주 = 비교적 건강하고 수월한 삶

## 조후 맛 조절법
- 한습 → 보충: 쓴맛(화)+매운맛(금) / 줄이기: 짠맛(수)+신맛(목)
- 난조 → 보충: 짠맛(수)+신맛(목) / 줄이기: 쓴맛(화)+매운맛(금)

## 조후 운동법
- 한습: 가벼운 유산소(걷기,조깅) → 화 기운 보충, 운동 후 보온 필수
- 난조: 수영(수 기운), 스트레칭(목 기운) → 열 식히기+유연성

## 신강(身强)/신약(身弱) 판정
1. 비겁+인성 vs 식상+재성+관성 세력 비교
2. 득령(월지), 득지(일지), 득세(연·시주) 확인
3. **최핵심 — 통근(通根)**: 천간 글자가 지지에 같은 오행(비겁) 또는 생해주는 오행(인성)을 두어 호응해야 실질적 힘 발휘
   - 통근 없는 천간 = "사주에 없는 것만 못함" (예: 인성 있어도 뿌리 없으면 공부머리·인내심 부족)
   - 일간이 지지에 뿌리 있으면(旺) → 주체성·자신감·지도력
   - 일간이 지지에 뿌리 없으면(衰) → 겉으로 강해 보여도 극신약, 타인 의존·눈치·자신감 결여
   - 월지·일지 모두 내 편이고 + 다른 자리에 하나 더 → 확실한 신강

## 신강/신약별 운동
- 신약: 명상, 요가, 필라테스 (에너지 소모 최소화)
- 신강(비겁 과다): 적극적 땀 흘리기 (축구, 달리기, 격투기 = 식상 발산)
- 수 부족 체질: 무리한 PT/중량운동 금지 (관절 약함), 부드러운 운동

## 신강 음식 관리 핵심 원리
- 신강이라고 모두 같은 음식이 아님 — 어떤 오행이 과다한지에 따라 다름
- 과다 오행의 맛 → 줄이기 / 부족 오행의 맛 → 챙기기
- **식상(일간이 生하는 오행)의 맛을 섭취**하면 에너지 발산에 도움
  - 목일간 → 식상=화 → 쓴맛 / 화일간 → 식상=토 → 단맛
  - 토일간 → 식상=금 → 매운맛 / 금일간 → 식상=수 → 짠맛
  - 수일간 → 식상=목 → 신맛
- 음식 + 땀 흘리는 운동 병행이 최선

## 신강/신약 건강 관리
- **신강 — 에너지 발산이 생존 루틴:**
  - 비겁 과다 → 축구, 격투기, 헬스(중량) 등 격렬한 운동으로 발산
  - 미발산 시: 오만/고집/급발진/집착/멘탈 와르르 붕괴/결정장애(인다신강)
  - 운동 외 발산법: 취미활동, 여행, 산책, 수다(타인과 교류), 연예인·프리랜서·형사 등 활동적 직업
  - "절대 지지 않겠다"는 투쟁심을 건전한 채널로 보내야 함
  - **실내 발산법**: 수다/전화(언어 표출), 글쓰기/그림/창작활동(예술 표현), 홈트레이닝(실내운동), 몰입형 취미 — 에너지를 속으로 삭이지 말 것
- **신약 — 에너지 보존이 핵심:**
  - 억눌린 감정 축적 → 우울, 공허감, 자기 학대
  - 격투기/헬스 → 기가 눌려 건강 해침 (금지)
  - 명상, 요가, 필라테스 → 차분히 심신 균형 다지기
  - 불리한 운(기구신 운)에는 삶의 규모 축소, 건강 최우선
- **공통:** 일지(日支)가 운에서 강한 충(3대1 쟁충 등) 맞으면 몸이 즉각적으로 망가짐 주의

## 오행별 직업 적성 (유저 제공)
- 목: 교육자(갑목=대학교수, 을목=초등교사), 체계/분류, 문과형
- 화: 디자인, 방송, 미디어, 무대, 시각예술, 문과형
- 토: 중재자, 신용/믿음 기반, 양측 포용/조율
- 금: 옳고그름 판별, 이성적 판단, 이과형 (금+수=이과 최적)
- 수: 학자, 지식인, 외국어/소리 관련, 이과형 (금+수=우뇌)
- 인사신 삼형살: 법무부, 군인, 경찰, 검찰, 교도관, 의사 → 흉→길 전환
- 비겁 과다 신강: 운동선수, 형사, 연예인, 프리랜서

**Why:** 기존 코드에 조후 분석 전무, 신강/신약에 따른 운동/건강 차별화 없었음
**How to apply:** analyzeJohu() 함수로 오행밸런스 기반 한습/난조 판정, 대운 건강 섹션에 조후 경고+식이+운동 자동 출력
